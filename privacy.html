개인정보처리방침

최종 수정일: 2026년 3월

1. 수집하는 정보

본 서비스(YT 자막 업로더)는 다음 정보에 접근합니다:

  • Google 계정 기본 정보 (이름, 이메일, 프로필 사진) — 로그인 확인용
  • YouTube 채널 접근 권한 — 자막 업로드 목적

2. 정보 사용 목적

수집된 정보는 오직 YouTube 자막 파일 업로드 기능 제공을 위해서만 사용됩니다. 마케팅, 광고, 제3자 제공 등의 목적으로 사용하지 않습니다.

3. 정보 저장

본 서비스는 서버에 사용자 정보를 저장하지 않습니다. 모든 처리는 브라우저 내에서 이루어지며, 페이지를 닫으면 액세스 토큰은 즉시 삭제됩니다.

4. 제3자 공유

사용자 정보를 제3자에게 판매하거나 공유하지 않습니다. YouTube API 사용은 Google의 서비스 약관을 따릅니다.

5. 권한 철회

언제든지 Google 계정 권한 설정에서 본 앱의 접근 권한을 철회할 수 있습니다.

6. 문의

개인정보 관련 문의는 서비스 운영자에게 연락해 주세요.

본 서비스는 Google LLC의 YouTube API Services를 사용합니다. Google 개인정보처리방침
